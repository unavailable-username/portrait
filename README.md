# Portrait
## An app used to display images with a description in the browser.

### Technologies Used
* Python 3.7
* Windows or Linux
* Flask 1.1.x

## Setup + Running (Linux)
Create a virtual env
```
python3 -m venv venv
. venv/bin/activate
pip install Flask
```

Run the application
```
export FLASK_APP=server.py
flask run
```

## Setup + Running (Windows)
Create a virtual env
```
py -3 -m venv venv
venv\Scripts\activate
pip install Flask
```

Run the application
```
set FLASK_APP=hello.py
flask run
```
